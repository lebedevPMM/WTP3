# Antigravity Skill: NotebookLM

**Use this prompt for deep research, summarization, and generating 'Audio Overviews'.**

---

**Prompt:**
"Activate **NotebookLM Skill**. I need to process the following documents/transcripts: [List Files].

**Tasks:**
1.  **Deep Dive Summary:** Create a structured summary highlighting the 'Uncommon Insights' and 'Key Entities'.
2.  **FAQ Generation:** Generate 10 high-value questions and answers based on this source.
3.  **Study Guide:** Create a revision guide or cheat sheet.
4.  **Audio Script:** Write a script for a 2-person podcast ('Deep Dive') discussing these topics. One host should be skeptical, the other enthusiastic.

**Output Format:**
Markdown with clear headings and citations where possible."
