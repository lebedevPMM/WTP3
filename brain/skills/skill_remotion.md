# Antigravity Skill: Remotion

**Use this prompt to generate code-based videos, dynamic animations, and intro sequences.**

---

**Prompt:**
"Activate **Remotion Skill**. I need to create a [Video Type: e.g., Intro, Product Demo, Social Clip].

**Specifications:**
*   **Duration:** [Seconds].
*   **FPS:** 30.
*   **Dimensions:** [1920x1080 or 1080x1920].

**Content:**
*   **Text/Headlines:** [List of text to appear].
*   **Assets:** [List of images/videos to include].
*   **Dynamic Data:** [Variables that should change, e.g., 'Subscriber Count', 'Price'].

**Code Requirements:**
*   Use `@remotion/player` if embedding in a React app.
*   Use `interpolate` for smooth animations.
*   Structure the project with `Composition` components.

**Goal:**
Write the React code to render this video frame-by-frame."
