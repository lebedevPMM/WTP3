# Antigravity Skills Library

This folder contains reusable prompts and templates for the core "Antigravity" skills.

## How to use
When you need to perform a specific task, open the corresponding skill file, copy the prompt, and paste it into your AI assistant (Cursor, Windsurf, etc.).

## The 5 Levels
1.  **[Image Generation](skill_image_generation.md)**: Create assets using DALL-E/Nano Banana.
2.  **[Presentations](skill_presentations.md)**: Build slide decks and stories.
3.  **[Animated Websites](skill_animated_website.md)**: Create high-end microsites with `framer-motion`.
4.  **[Infographics](skill_infographics.md)**: Visualize data programmatically.
5.  **[Documents](skill_documents.md)**: Generate specs, reports, and manuals.

---
*Tip: Combine these skills with the [Blast Master Prompt](../blast_master_prompt.md) for best results.*
