# Antigravity Skill: Animated Websites

**Use this prompt to build high-end, interactive web experiences (Microsites).**

---

**Prompt:**
"I want to build a **Unicorn** website (Great Design + Great Functionality).

**Core Request:**
Build a responsive, animated landing page for [Product/Service].

**Design System:**
- **Theme:** [e.g., Midnight Sky, Glassmorphism, Minimalist]
- **Animations:**
    - Use `framer-motion` (React) or GSAP.
    - Implement smooth scroll and hover effects.
    - Add micro-interactions on all buttons.
- **Typography:** Use modern Google Fonts (e.g., Inter, Outfit).

**Constraint:**
Do not use generic colors. Use a curated palette (e.g., HSL formatted). Ensure the design feels 'alive' and premium."
