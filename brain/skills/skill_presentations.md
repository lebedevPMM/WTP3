# Antigravity Skill: Presentations

**Use this prompt to generate slide decks or presentation materials.**

---

**Prompt:**
"I need a presentation deck for [Topic/Meeting].

**Structure:**
1.  **Title Slide:** catchy title and subtitle.
2.  **Problem Statement:** Clear definition of the issue.
3.  **Solution:** How [Product/Idea] solves it.
4.  **Benefits:** Key value propositions (bullet points).
5.  **Call to Action:** Next steps.

**Format:**
- Generate this as a **React-based slide deck** (using a library like `reveal.js` or custom components) OR as a downloadable Markdown/PDF summary.
- Focus on visual hierarchy and clean layout."
